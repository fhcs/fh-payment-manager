# laravel-package-skeleton

## Introduction

Laravel Payment system manager

## Installation

### Composer

```shell script
composer require vladmeh/payment-manager
```

or add the following to your requirement part within the composer.json:

```json
{
    "require": {
        "vladmeh/payment-manager": "^1.*"
    }
}
```
